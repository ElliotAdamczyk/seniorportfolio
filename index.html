Solstice

See the Github Repository

Solstice is the website/game my team made during the first ever Pasadena High School App Academy hack-a-thon. It is a single player website based game that aims to eliminate summer boredom by presenting different things to do. Some of these activities are playing a beach based pong, reading books, and draw. We won the best original idea for the hack-a-thon.
